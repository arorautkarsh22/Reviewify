Reviewify is a student-centric feedback analysis platform built to amplify the voices of students and help future students make informed decisions about colleges and courses. Our system collects reviews through Google Forms and uses advanced Natural Language Processing (NLP) techniques to analyze sentiments and generate visual summaries like word clouds. The project is also equipped with a modern, responsive user interface focused on usability and accessibility.

The core goal of Reviewify is to bridge the information gap between current and prospective students, offering transparent, real-time, and anonymous insights based on collective student experience.

This project combines Machine Learning, Data Analysis, and Frontend Development into a single seamless platform—crafted entirely by students, for students.
